# 🛒 Ecommerce API - FastAPI

API backend de un ecommerce construida con FastAPI, MySQL y arquitectura limpia.
Este proyecto forma parte de una serie donde se construye paso a paso desde cero.

## 🚀 Tecnologías

- FastAPI
- MySQL
- SQLAlchemy
- JWT Authentication
- Docker (próximamente)

## 📁 Estructura

app/
│── api/          # Rutas
│── models/       # Modelos DB
│── schemas/      # Validaciones
│── services/     # Lógica de negocio
│── db/           # Conexión DB
│── core/         # Configuración

## ⚙️ Instalación

git clone https://github.com/Nezareth07/ecommerce.git
cd ecommerce
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
